Thank you for your interest in contributing to Hack Client for 1.21.1 (NeoForge)! ✅

How to contribute
- Open an issue to discuss major changes or report bugs.
- Send a pull request against `main` with a clear description of the change.
- Keep commits focused and small; write a short commit message that explains the intent.

Development checklist
- Run the project build locally: `./gradlew build` and ensure tests pass (if any).
- Keep code style consistent with the project's existing source.

Contact
- If you'd rather discuss contributions or need to reach me directly, email: **kenzo101studios@gmail.com**

Thanks — your help is appreciated!
